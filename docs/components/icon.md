# Icon 图标
常用的图标集合。


## 使用方法
通过icon属性值'icon-iconName'来使用

<ClientOnly>
  <icon-demo></icon-demo>
</ClientOnly>

```vue
<wl-icon icon="icon-edit"></wl-icon>
<wl-icon icon="icon-delete"></wl-icon>
<wl-icon icon="icon-upload"></wl-icon>
<wl-button type="primary" icon="icon-search">搜索</wl-button>
```
